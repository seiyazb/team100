## 1. はじめに

### 1.1 TalentOSとは

TalentOSは、IT人材のスキル情報をAIヒアリングにより自動収集・整理し、営業担当者が最適な人材を検索・マッチングできる人材管理システムです。

### 1.2 主な機能

| 機能 | 概要 | 利用可能ロール |
|------|------|----------------|
| AIヒアリング | AIチャットによるスキル情報の自動収集 | エンジニア、管理者 |
| スキルシート | スキル情報の閲覧・編集・PDF出力 | エンジニア、営業、管理者 |
| 人材検索 | 自然言語によるエンジニア検索 | 営業、管理者 |
| ユーザー管理 | ユーザーの追加・一覧表示 | 管理者 |

### 1.3 動作環境

- ブラウザ：Google Chrome（推奨）、Microsoft Edge、Firefox
- URL：`http://localhost:8000`（開発環境）

### 1.4 ロール（役割）について

本システムには3つのロールがあります。ロールによってアクセスできる画面が異なります。

| ロール | 説明 | アクセス可能な画面 |
|--------|------|-------------------|
| engineer（エンジニア） | スキル情報を登録するIT技術者 | AIヒアリング、スキルシート |
| sales（営業） | 人材を検索・提案する営業担当者 | 人材検索、スキルシート（閲覧） |
| admin（管理者） | システム全体を管理する管理者 | 全画面 |

## 2. ログイン・ログアウト

### 2.1 ログイン

**画面：** ログイン画面（`/login`）

<!-- 📸 スクリーンショット：ログイン画面 -->
> [スクリーンショット挿入位置] ログイン画面の全体表示

**操作手順：**

1. ブラウザでシステムのURL（`http://localhost:8000`）にアクセスします
2. ログイン画面が表示されます
3. 「ユーザーID」欄にユーザーIDを入力します
4. 「パスワード」欄にパスワードを入力します
5. 「ログイン」ボタンをクリックします
6. ログインに成功すると、ロールに応じた画面に自動遷移します
   - エンジニア → AIヒアリング画面
   - 営業 → 人材検索画面
   - 管理者 → ユーザー管理画面

<!-- 📸 スクリーンショット：ログインエラー時の表示 -->

| エラーメッセージ | 原因 | 対処法 |
|-----------------|------|--------|
| ユーザーIDまたはパスワードが正しくありません | ID/パスワードの入力ミス | 正しい情報を再入力してください |

### 2.2 ログアウト

**操作手順：**

1. 画面左側のサイドメニュー下部にある「ログアウト」ボタンをクリックします

## 3. AIヒアリング（エンジニア向け）

**対象ロール：** エンジニア、管理者

**画面：** AIヒアリング画面（`/hearing`）

### 3.1 画面構成

<!-- 📸 スクリーンショット：AIヒアリング画面の全体レイアウト（番号付き注釈） -->
> [スクリーンショット挿入位置] AIヒアリング画面の全体レイアウト

AIヒアリング画面は以下の要素で構成されています：

| 番号 | 要素 | 説明 |
|------|------|------|
| ① | テーマ進捗インジケーター | 現在のヒアリングテーマと進捗状況を表示 |
| ② | チャットエリア | AIとの会話が表示される領域 |
| ③ | メッセージ入力欄 | AIへの回答を入力するテキストエリア |
| ④ | プロフィールカード | エンジニアの基本情報を表示 |
| ⑤ | 抽出データプレビュー | ヒアリングから抽出された構造化データ |

### 3.2 ヒアリングの流れ

AIヒアリングは以下の3つのテーマで段階的に進行します：

```
基本情報（Basic） → 経歴（Career） → スキル（Skills）
```

各テーマで4〜5問のやり取りを行い、完了すると次のテーマに進みます。

### 3.3 ヒアリングを開始する

**操作手順：**

1. サイドメニューから「AIヒアリング」をクリックします
2. AIヒアリング画面が表示されます
3. テーマ進捗インジケーターで現在のテーマ（最初は「基本情報」）を確認します

<!-- 📸 スクリーンショット：テーマ進捗インジケーター -->
> [スクリーンショット挿入位置] テーマ進捗の表示（Basic → Career → Skills）

### 3.4 AIと会話する

**操作手順：**

1. AIからの質問がチャットエリアに表示されます
2. メッセージ入力欄に回答を入力します
3. 送信ボタンをクリック（またはEnterキー）で回答を送信します
4. AIの応答がリアルタイムでストリーミング表示されます
5. 次の質問が表示されたら、同様に回答します

<!-- 📸 スクリーンショット：チャットのやり取り例 -->
> [スクリーンショット挿入位置] AIとの会話例（質問と回答）

**ポイント：**
- できるだけ具体的に回答すると、より正確なスキルシートが生成されます
- 技術スタックや経験年数は具体的な名称・数値で回答してください
- ヒント表示（チップ）が表示される場合は、参考にして回答できます

### 3.5 テーマの完了と次のテーマへの移行

**操作手順：**

1. 各テーマで必要な質問への回答が完了すると、テーマが「完了」状態になります
2. テーマ進捗インジケーターが更新されます
3. 自動的に次のテーマのヒアリングが開始されます

<!-- 📸 スクリーンショット：テーマ完了時の表示 -->
> [スクリーンショット挿入位置] テーマ完了時の進捗表示

### 3.6 抽出データの確認

ヒアリングの各テーマが完了すると、右側のパネルに抽出されたデータが表示されます。

<!-- 📸 スクリーンショット：抽出データプレビュー -->
> [スクリーンショット挿入位置] 右パネルの抽出データ表示

## 4. スキルシート管理

**対象ロール：** エンジニア、営業、管理者

**画面：** スキルシート画面（`/skillsheet`）

### 4.1 画面構成

<!-- 📸 スクリーンショット：スキルシート画面の全体レイアウト -->
> [スクリーンショット挿入位置] スキルシート画面の全体レイアウト

| 番号 | 要素 | 説明 |
|------|------|------|
| ① | AIチャットエリア（左） | AIに指示してスキルシートを編集できるチャット |
| ② | スキルシートプレビュー（右） | 現在のスキルシート内容をリアルタイム表示 |
| ③ | アクションボタン | 保存・PDF出力・プレビューボタン |

### 4.2 スキルシートを閲覧する

**操作手順：**

1. サイドメニューから「スキルシート」をクリックします
2. 自分のスキルシート（エンジニアの場合）または指定エンジニアのスキルシートが表示されます

<!-- 📸 スクリーンショット：スキルシートプレビュー表示 -->
> [スクリーンショット挿入位置] スキルシートの各セクション表示

**スキルシートの構成：**

| セクション | 内容 |
|-----------|------|
| 基本情報 | 氏名、専門分野、勤務地、最寄駅、転勤可否 |
| 学歴 | 学歴レベル、学校名、学部・学科 |
| 資格 | 保有資格一覧 |
| 語学 | 語学スキル |
| 技術スキル | 使用可能な技術・ツール |
| 自己PR | 自己PR文 |
| 経歴 | プロジェクト経歴（期間、チーム規模、役割、技術スタック） |

### 4.3 スキルシートを編集する

**操作手順：**

1. 左側のAIチャットエリアで、変更したい内容をAIに指示します
   - 例：「自己PRを更新してください」「Pythonの経験年数を5年に変更してください」
2. AIが内容を反映し、右側のプレビューが更新されます
3. 内容を確認し、「保存」ボタンをクリックして保存します

<!-- 📸 スクリーンショット：AIチャットでの編集指示例 -->
> [スクリーンショット挿入位置] チャットでの編集指示と反映

### 4.4 スキルシートの最適化

AIによるスキルシートの文章最適化を行えます。

**操作手順：**

1. スキルシート画面で「最適化」ボタンをクリックします
2. AIがスキルシート全体の文章を改善・最適化します
3. 最適化結果がプレビューに反映されます
4. 内容を確認し、「保存」ボタンをクリックして保存します

<!-- 📸 スクリーンショット：最適化前後の比較 -->
> [スクリーンショット挿入位置] 最適化前と最適化後の表示

### 4.5 PDF出力

**操作手順：**

1. スキルシート画面で「PDF出力」ボタンをクリックします
2. PDFファイルが自動的にダウンロードされます
3. ファイル名は `skillsheet_{エンジニアID}.pdf` の形式です

<!-- 📸 スクリーンショット：PDF出力ボタンの位置 -->

### 4.6 印刷プレビュー

**操作手順：**

1. スキルシート画面で「プレビュー」ボタンをクリックします
2. ブラウザの印刷プレビュー画面が表示されます
3. 必要に応じてそのまま印刷できます

## 6. ユーザー管理（管理者向け）

### 6.2 ユーザー一覧を確認する

**操作手順：**

1. サイドメニューから「ユーザー管理」をクリックします
2. 登録済みユーザーの一覧がテーブル形式で表示されます

<!-- 📸 スクリーンショット：ユーザー一覧テーブル -->
> [スクリーンショット挿入位置] ユーザー一覧テーブルの表示

**テーブルの項目：**

| 列 | 説明 |
|----|------|
| ユーザーID | ログインに使用するID |
| 名前 | ユーザーの表示名 |
| ロール | ユーザーの役割（バッジで色分け表示） |

### 6.3 新規ユーザーを追加する

**操作手順：**

1. 「ユーザー追加」ボタンをクリックします
2. ユーザー追加モーダルが表示されます

| 項目 | 入力内容 | 必須 |
|------|---------|------|
| ユーザーID | ログインに使用するID（半角英数字） | ○ |
| 名前 | ユーザーの表示名 | ○ |
| パスワード | ログインパスワード | ○ |
| ロール | engineer / sales / admin から選択 | ○ |

4. 入力内容を確認し、「作成」ボタンをクリックします
5. ユーザーが作成され、一覧テーブルに追加されます

### 7.3 画面遷移図

```
ログイン画面 (/login)
    │
    ├── [engineer] ──→ AIヒアリング (/hearing)
    │                        │
    │                        └──→ スキルシート (/skillsheet)
    │                                  │
    │                                  ├── 保存
    │                                  ├── PDF出力
    │                                  └── 最適化
    │
    ├── [sales] ────→ 人材検索 (/search)
    │                        │

    │
    └── [admin] ────→ ユーザー管理 (/users)
                             │
                             ├── ユーザー追加
                             └── 全画面アクセス可能
```

### 7.4 トラブルシューティング

| 症状 | 考えられる原因 | 対処法 |
|------|--------------|--------|
| ログインできない | ID/パスワードの入力ミス | 正しい情報を確認して再入力 |
| 画面が表示されない | セッション期限切れ（8時間） | 再度ログインしてください |
| 「アクセス権限がありません」と表示される | ロールに対する権限不足 | 管理者にロールの確認を依頼 |
| AIの応答が遅い | Dify APIへの接続遅延 | しばらく待つか、ページを再読み込み |
| PDF出力できない | weasyprint未インストール | 管理者にサーバー設定を確認依頼 |
| 検索結果が0件 | 該当するエンジニアが未登録 | 検索条件を変えて再検索 |
